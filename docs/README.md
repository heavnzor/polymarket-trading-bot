# Documentation Index

## Scope

This repository runs a hybrid trading system:

- Market Making (fast loop, multi-market)
- Crypto Directional (Student-t model on BTC/ETH thresholds)
- Django control-plane + Next.js dashboard

## Folders

- `docs/architecture/`:
  service boundaries and runtime design.
- `docs/api/`:
  REST and websocket contracts.
- `docs/operations/`:
  deployment and runtime operations.

Obsolete migration docs from the legacy 9-agent pipeline were removed.
